# Intake Context

DealSniperAI is being built as a Phase 1 MVP.

The Phase 1 scope is:

```text
Ingest business
Map signals
Interpret context
Score opportunity
Store result
Display profile
```

Phase 1 must not send external owner communications.

Core architecture:

```text
RIOS = Research → Intelligence → Opportunity → Strategy → Execution
ICM  = Data → Context → Signal → Interpretation → Score → Action
MWP  = Numbered filesystem workspaces with review gates
```
